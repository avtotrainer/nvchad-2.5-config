# ­ЪЊЂ NvimTree Shortcut Cheat Sheet 

## ­ЪЊЂ рЃњрЃљрЃ«рЃАрЃюрЃљ / рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `<CR>`, `o`  | рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ                             |
| `<Tab>`      | рЃцрЃљрЃўрЃџрЃўрЃА рЃърЃарЃћрЃЋрЃўрЃБ (рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЋрЃљ)                 |
| `<C-V>`      | рЃњрЃљрЃ«рЃАрЃюрЃљ рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃа рЃњрЃљрЃДрЃЮрЃцрЃљрЃерЃў                |
| `<C-X>`      | рЃњрЃљрЃ«рЃАрЃюрЃљ рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃа рЃњрЃљрЃДрЃЮрЃцрЃљрЃерЃў              |
| `<C-T>`      | рЃњрЃљрЃ«рЃАрЃюрЃљ рЃљрЃ«рЃљрЃџ рЃбрЃљрЃЉрЃерЃў                         |
| `-`          | рЃќрЃћрЃЏрЃЮрЃЌ (рЃЏрЃерЃЮрЃЉрЃћрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў)               |
| `P`          | рЃЏрЃерЃЮрЃЉрЃћрЃџ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ              |
| `R`          | рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃ«рЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ                    |

## ­ЪЊѓ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃў
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `<BS>`       | рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ                       |
| `E`          | рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ                  |
| `W`          | рЃДрЃЋрЃћрЃџрЃљ рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ                 |
| `L`          | рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ/рЃЊрЃљрЃЏрЃљрЃџрЃЋрЃљ     |

## ­ЪћЇ рЃцрЃўрЃџрЃбрЃарЃћрЃЉрЃў
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `I`          | Git ignored рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ/рЃЊрЃљрЃЏрЃљрЃџрЃЋрЃљ      |
| `H`          | рЃцрЃљрЃарЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃбрЃЮрЃњрЃџрЃў (dotfiles)          |
| `C`          | Git clean рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃбрЃЮрЃњрЃџрЃў                  |
| `U`          | Hidden рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃбрЃЮрЃњрЃџрЃў                     |
| `F`          | рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃцрЃўрЃџрЃбрЃарЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ               |
| `f`          | рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў рЃцрЃўрЃџрЃбрЃарЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ                   |

## ­ЪЊЮ рЃерЃћрЃЦрЃЏрЃюрЃљ / рЃгрЃљрЃерЃџрЃљ / рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `a`          | рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃю рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ        |
| `d`          | рЃгрЃљрЃерЃџрЃљ                                     |
| `r`          | рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ                                 |
| `e`          | рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃљрЃ«рЃћрЃџрЃў)                |
| `u`          | рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ рЃАрЃарЃБрЃџрЃў рЃЉрЃўрЃџрЃўрЃЎрЃўрЃЌ                  |
| `<C-R>`      | рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ                 |

## Рюѓ№ИЈ рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ / рЃЕрЃљрЃАрЃЏрЃљ
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `c`          | рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ                                   |
| `x`          | рЃљрЃЏрЃЮрЃГрЃарЃљ                                     |
| `p`          | рЃЕрЃљрЃАрЃЏрЃљ                                      |
| `y`          | рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ                    |
| `gy`         | рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃЉрЃўрЃџрЃўрЃЎрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ                |
| `ge`         | рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃцрЃљрЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ (basename)  |
| `Y`          | рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌрЃў рЃЉрЃўрЃџрЃўрЃЎрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ                |

## ­Ъћќ рЃЉрЃБрЃЦрЃЏрЃљрЃарЃЎрЃћрЃЉрЃў / рЃюрЃљрЃњрЃљрЃЋрЃў
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `m`          | рЃЉрЃБрЃЦрЃЏрЃљрЃарЃЎрЃўрЃА рЃбрЃЮрЃњрЃџрЃў                           |
| `bd`         | рЃЉрЃБрЃЦрЃЏрЃљрЃарЃЎрЃћрЃЉрЃўрЃА рЃгрЃљрЃерЃџрЃљ                         |
| `bt`         | рЃЉрЃБрЃЦрЃЏрЃљрЃарЃЎрЃћрЃЉрЃўрЃА рЃюрЃљрЃњрЃљрЃЋрЃерЃў рЃњрЃљрЃЊрЃљрЃњрЃЊрЃћрЃЉрЃљ             |
| `bmv`        | рЃЉрЃБрЃЦрЃЏрЃљрЃарЃЎрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃљ                      |
| `D`          | рЃцрЃљрЃўрЃџрЃўрЃА рЃюрЃљрЃњрЃљрЃЋрЃерЃў рЃњрЃљрЃЊрЃљрЃњрЃЊрЃћрЃЉрЃљ                  |

## ­ЪћД рЃАрЃ«рЃЋрЃљ
| рЃЎрЃџрЃљрЃЋрЃўрЃерЃў      | рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ                                |
|--------------|-------------------------------------------|
| `q`          | `nvim-tree`-рЃўрЃА рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ                    |
| `.`          | рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ                       |
| `g?`         | рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃўрЃБ                           |
| `s`          | рЃЊрЃљрЃџрЃљрЃњрЃћрЃЉрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃЌ                         |
| `S`          | рЃФрЃћрЃЉрЃюрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃерЃў                            |
| `[c`, `]c`   | рЃгрЃўрЃюрЃљ / рЃерЃћрЃЏрЃЊрЃћрЃњрЃў Git рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ              |
| `[e`, `]e`   | рЃгрЃўрЃюрЃљ / рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЊрЃўрЃљрЃњрЃюрЃЮрЃАрЃбрЃўрЃЎрЃљ               |
| `J`, `K`     | рЃЉрЃЮрЃџрЃЮ / рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃерЃЋрЃўрЃџрЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ           |
| `<`, `>`     | рЃгрЃўрЃюрЃљ / рЃерЃћрЃЏрЃЊрЃћрЃњрЃў sibling-рЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ       |
